这是我的**个人数字庭院**——一个存放思考、实验与信念的地方。

## 🛠️ 技术栈（极简主义践行者）

| 层级 | 选型 | 理由 |
|------|------|------|
| **前端** | 原生 HTML + CSS | 零依赖、秒开、无障碍友好、10 年后仍可读 |
| **部署** | GitHub Pages | 免费、自动化、与代码同源 |
| **风格** | 橘色系 + 响应式 | 视觉温暖，暗黑模式自适应，手机/电脑皆宜 |
| **哲学** | KISS（Keep It Simple, Stupid） | 简单，是长期主义的起点 |

> 💡 *“复杂常是懒惰的遮羞布，简单才是深思的勋章。”*
